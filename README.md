# Case Intuitive

Este repositório contém soluções para quatro questões específicas, organizadas nos seguintes diretórios:

- `questao_um`: Solução da primeira questão.
- `questao_dois`: Solução da segunda questão.
- `questao_tres`: Solução da terceira questão.
- `questao_quatro`: Solução da quarta questão.

## Dependências

As dependências necessárias para executar as soluções estão listadas no arquivo `requirements.txt`. Para instalá-las, execute:

```bash
pip install -r requirements.txt
```

## Como Executar

### Questão 1

Navegue ate o diretorio do codigo fonte (src) e execute:
```bash
python main.py
```

### Questão 2
Navegue ate o diretorio do codigo fonte (src) e execute:
```bash
python main.py
```
### Questão 3

Navegue ate o diretorio do codigo fonte (src) e execute:
```bash
python ajuste_encoding.py
```

Após isso, utilize cria_tabelas.sql para criar suas tabelas em sql

Utilize importa_tabelas.sql para importar os dados do csv's para as tabelas

Utilize queries.sql para fazer as consultas necessárias

### Questão 4

Navague até o diretório backend para rodar o servdior que contem a api do programa e dê o comando para ativa-la

```bash
python app.py
```

Navague até o diretório frontend/operadoras-web e execute o comando para rodar o frontend

```bash
npm run serve
```

Após isso, basta entrar no link local que irá aparecer no terminal e acessar o Buscador de operadoras web
